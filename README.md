# La Fábrica: OpenClaw 🚀

Un proyecto para reacondicionar y dar una segunda vida a teléfonos antiguos convirtiéndolos en servidores IA personales y autónomos con **OpenClaw**. Sin cloud. Sin suscripciones. Tu propio cerebro digital de bolsillo.

## 🛠️ ¿Qué puedes conseguir?

- **Chat por Voz**: Habla con el dispositivo reacondicionado y recibe respuestas inteligentes con voz.
- **Visión IA**: Utiliza la cámara del teléfono y la IA te describirá lo que ve.
- **Avatar Animado**: Una cara de neón integrada que parpadea y habla respondiendo a tus dudas.
- **IA 100% Offline**: Funciona sin internet usando modelos locales con Ollama y Whisper.
- **Servidor 24/7**: El dispositivo funciona como un servidor local accesible de forma remota (Telegram, API, etc.).

## 📋 Requisitos de La Fábrica

- Un teléfono Android antiguo (recomendado mínimo 3GB de RAM, 4GB+ para modo offline).
- **Restablecimiento de fábrica**. No uses tu dispositivo principal ni cuentas con tus datos personales por seguridad.
- Conexión por cable USB a tu PC/Mac para cargar la configuración de OpenClaw.
- Instalar Termux versión F-Droid en el terminal Android.

## 🚀 Instalación y Despliegue

1. Conecta el dispositivo Android por USB con Depuración USB habilitada.
2. Abre la terminal en tu odenador y ejecuta el siguiente comando:
   ```bash
   cd ClawMobil && bash scripts/deploy.sh
   ```
3. ¡Listo! El entorno se desempaquetará y el dispositivo cobrará vida.
